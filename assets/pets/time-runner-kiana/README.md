# 时砾逐光

琪亚娜非官方同人 Codex V2 桌宠。

## 原生包能力

- 九组 Codex 标准状态动画
- 十六方向鼠标视线跟随
- 适用于支持 Codex V2 自定义桌宠格式的客户端
- 静态资源包，不包含可执行代码

本包不包含 Windows 增强版中的工作状态自动奔跑、待机随机彩蛋、特殊悬浮与拖动节奏或主题换肤逻辑。

## 安装

将本目录复制到：

```text
~/.codex/pets/time-runner-kiana/
```

然后在 `Codex 设置 > 外观 > 宠物` 中刷新并选择“时砾逐光”。也可以通过 [CodexPet 公开页面](https://codexpet.xyz/zh/pets/community/time-runner-kiana/) 安装。
