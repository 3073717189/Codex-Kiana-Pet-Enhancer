# Codex 琪亚娜增强桌宠 v1.0.3

这是一次安装兼容性补丁。桌宠素材和动作没有变化。

## 本次修复

- 修复 Codex 更新后旧安装目录已被移除、但 `state.json` 仍存在时，安装器误报“旧桌宠运行状态”的问题
- 恢复脚本缺失时，安装器会核验旧 Codex 和注入进程的真实身份
- 确认旧运行状态已经失效后，会先归档状态文件再继续覆盖安装
- PID 已被其他程序复用或旧端口被无关监听占用时，不会结束无关进程
- 保留 v1.0.2 的空闲端口自动回退与详细错误日志修复

## 升级

已安装 v1.0.0、v1.0.1 或 v1.0.2 的用户无需先卸载：

1. 下载并完整解压 `Codex-Kiana-Pet-Enhancer-v1.0.3-win-x64.zip`。
2. 完全退出所有 Codex 窗口。
3. 双击 `安装 Codex 琪亚娜增强桌宠.cmd` 覆盖升级。
4. 以后继续使用“Codex 琪亚娜桌宠”快捷方式启动。

原生“时砾逐光”宠物、当前宠物选择和安装前备份都会保留。

## 功能

- Codex 工作时自动奔跑，并合并多个任务的工作状态
- 鼠标视线跟随、待机随机彩蛋和原生拖拽兼容
- 等待、审核与失败状态动画
- 失败动画减速并停留末帧
- 安装、升级和卸载均保留回滚路径；已有同名桌宠会先备份
