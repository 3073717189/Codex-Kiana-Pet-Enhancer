# Codex 琪亚娜增强桌宠 v1.0.2

这是一次启动兼容性补丁。桌宠素材和动作没有变化。

## 本次修复

- 修复 Codex 自动更新后，旧调试会话可能残留占用 `9345`，导致增强桌宠启动失败的问题
- 启动快捷方式不再锁死单一端口；首选端口不可用时会自动选择新的本机回环端口
- 卸载入口会读取桌宠实际使用的端口
- 启动失败时会在弹窗和日志中显示真实原因，方便反馈和排查

## 升级

已安装 v1.0.0 或 v1.0.1 的用户无需先卸载：

1. 下载并完整解压 `Codex-Kiana-Pet-Enhancer-v1.0.2-win-x64.zip`。
2. 完全退出所有 Codex 窗口。
3. 双击 `安装 Codex 琪亚娜增强桌宠.cmd` 覆盖升级。
4. 以后继续使用“Codex 琪亚娜桌宠”快捷方式启动。

原生“时砾逐光”宠物、当前宠物选择和安装前备份都会保留。

## 功能

- Codex 工作时自动奔跑，并合并多个任务的工作状态
- 鼠标视线跟随、待机随机彩蛋和原生拖拽兼容
- 等待、审核与失败状态动画
- 失败动画减速并停留末帧
- 安装、升级和卸载均保留回滚路径；已有同名桌宠会先备份
