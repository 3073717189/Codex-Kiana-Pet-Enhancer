# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/)。

## [1.0.2] - 2026-07-21

### Fixed

- 修复 Codex 自动更新后旧调试会话可能残留占用首选端口，导致增强桌宠快捷方式启动失败的问题
- 默认快捷方式不再锁死 `9345`，首选端口不可用时会安全切换到下一个空闲的本机回环端口
- 卸载入口默认读取实际运行端口，同时保留开发者显式固定端口的能力
- 启动器现在会记录并显示 PowerShell 的真实错误输出，不再只报告退出代码 `1`

## [1.0.1] - 2026-07-20

### Fixed

- 修复鼠标视线跟随朝向右下方时，`112.5°`、`135°`、`157.5°` 三个方向错误显示为左下方的问题
- 重新整理视线跟随图集第 9 行，保持八个方向之间的动作连续性；其余十行动画素材不变

## [1.0.0] - 2026-07-19

### Added

- Windows 10/11 x64 纯桌宠安装包
- “时砾逐光”琪亚娜自定义桌宠
- Codex 工作状态、多任务状态合并和自动奔跑
- 鼠标视线跟随、待机彩蛋和原生拖拽兼容
- 审核、等待和失败状态动画
- 失败动画减速及末帧停留
- 桌面与开始菜单增强启动快捷方式
- 可回滚安装与卸载；保护安装前已有的同名桌宠
- 内置并校验官方 Node.js 22.23.1，无需用户预装运行环境
- 软件、上游代码、第三方组件和角色素材的独立许可说明

### Security

- 调试会话仅监听 `127.0.0.1`
- 不修改 `WindowsApps`、`app.asar`、应用签名或 `.codex\config.toml`
- 发布清单为 payload 中的每个文件记录 SHA-256
